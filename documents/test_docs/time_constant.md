# Time Constant in First-Order Systems

The time constant is a parameter that describes how quickly a first-order dynamic system responds to a step input. It is usually written as tau and has units of time.

For a first-order system, the step response reaches about 63.2 percent of its final value after one time constant. After about five time constants, the response is usually considered close to steady state.

In an RC circuit, the time constant is:

```text
tau = R * C
```

where R is resistance in ohms and C is capacitance in farads. A larger time constant means a slower response, while a smaller time constant means a faster response.

Time constants are also used in control systems, thermal systems, and exponential smoothing to describe exponential decay or exponential approach to a final value.
